# 📘 **Eng Conversation Server**  
Django를 활용한 영어 회화 연습 웹 애플리케이션입니다.
 
## 📋 **기능**  
- 사용자 계정 생성 및 관리  
- 실시간 채팅 기능  
- 영어 회화 연습 전용 섹션 제공
